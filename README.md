# projekt_strona_api_CEPIK
Strona wyszukuje dane pojadów zarejestrowanych w Polsce i grupuje je ze względu na rok rejestracji oraz tworzy pliki z tabelami danych. 
Strona internetowa prosi o dane do logowania (login: admin, hasło: admin123). Po zalogowaniu formularz pyta o rok z którego chcemy pobrać dane. 
Aplikacja pobiera dane grupując je na województwa i zapisuje je w plikach csv w postaci tabeli.
